# BALLOTIQ
Blockchain-based voting system
BALLOTIQ is a blockchain-based voting platform designed to ensure security, transparency, decentralization, and voter privacy.
Only the admin has permission to view and publish the final election results, while voter identity remains fully anonymous on the blockchain.

FEATURES:-
Security & Blockchain

Decentralized vote recording using blockchain

Each vote stored as an immutable block

Voter identity is never linked to the candidate

VOTING SYSTEM :-

Voter registration & login

Unique voter authentication

One vote per user enforced programmatically

Admin-controlled result visibility

ADMIN DASHBOARD :-

View total votes per candidate

Publish/Unpublish results to the public portal

Manage candidates & voters
